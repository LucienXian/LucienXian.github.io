command line options

先来看两个基本的选项：

–version show program’s version number and exit
-h, –help show help about the command line options

接下来介绍一个message control的命令后选项：–disable和–enable，如果我们只是希望启动某些checker，可以先用–disable = all然后使用–enable = \，其中\是逗号分隔的检查器名称和消息符号列表。这样的使用方式对于仅仅希望启动少数的checkers非常有用。
